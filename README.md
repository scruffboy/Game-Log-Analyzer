# Game Log Analyzer

Инструмент для автоматизированного сбора, парсинга и хранения игровых логов на базе Python и SQLAlchemy.

## Основные возможности
- **ETL-конвейер**: Автоматическое чтение сырых текстовых логов и трансформация в объекты Python.
- **ORM Интеграция**: Использование SQLAlchemy для работы с реляционными БД (SQLite/PostgreSQL).
- **Типизация данных**: Строгая типизация через Dataclasses и Type Hints.
- **Универсальность**: Динамическое управление путями через `pathlib` и централизованный конфиг.

## Технологический стек
- **Python 3.10+**
- **SQLAlchemy 2.0**
- **SQLite**
- **Pathlib**
